# Cohesion extensions

## Installing extensions
To install an extension, simply copy the link of the extension on the table below, then, in Cohesion, go to Settings > Extensions > Add Extension, paste the link and click Okay. The extension will be added to your Cohesion instance.

Extensions can be updated by clicking the "Update" button on the extension card. This will fetch the latest version of the extension from the source.
